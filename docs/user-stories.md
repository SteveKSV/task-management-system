# Task Management System - User Stories

### Projects Management

1. As a user, I want to create a project so that I can organize tasks by topic.
2. As a user, I want to edit a project name and description.
3. As a user, I want to delete a project that I no longer need.
4. As a user, I want to view a list of my projects.
5. As a user, I want to access only my own projects.

### 

### Authentication \& Users

1. As a user, I want to register an account so that I can access the system.
2. As a user, I want to log in using email and password so that I can securely access my projects.
3. As a user, I want to log out so that my session is safely terminated.
4. As a user, I want my password to be securely stored and encrypted.
5. As a user, I want to stay authenticated using JWT tokens.



### Boards \& Columns (Kanban)

1. As a user, I want to view a Kanban board for a project.
2. As a user, I want the board to contain default columns (To Do, In Progress, Done).
3. As a user, I want to reorder columns if needed.
4. As a user, I want to move tasks between columns using drag-and-drop.
5. As a user, I want column changes to be saved automatically.



### Task Cards

1. As a user, I want to create a task card within a project.
2. As a user, I want to edit task details (title, description).
3. As a user, I want to assign a priority to a task.
4. As a user, I want to set a due date for a task.
5. As a user, I want to delete a task card.



### Task Status \& Completion

1. As a user, I want to move a task between columns to update its status.
2. As a user, I want to mark a task as completed.
3. As a user, I want completed tasks to be visually distinguished.
4. As a user, I want to reopen a completed task if needed.



### Task Details \& Interaction

1. As a user, I want to add comments to a task.
2. As a user, I want to edit or delete my own comments.
3. As a user, I want to see task creation and update timestamps.
4. As a user, I want to attach additional information to tasks.



### Filtering \& Search

1. As a user, I want to filter tasks by status.
2. As a user, I want to filter tasks by priority.
3. As a user, I want to search tasks by title or description.
4. As a user, I want to sort tasks by due date.



### Security \& Permissions

1. As a user, I want access to be restricted to my own projects.
2. As a user, I want unauthorized users to be denied access.
3. As a user, I want my data to be securely stored.
4. As an admin, I want to manage users (optional).
